# User Stories — Leśny Herbarium

## ICP 1: Uczeń (Kasia, 17 lat)
> Licealistka przygotowująca się do matury z biologii. Szuka interaktywnych materiałów do nauki o ekosystemach.

| # | User Story | Priorytet | Status |
|---|---|---|---|
| US-1.1 | Jako uczeń, chcę przeglądać katalog roślin leśnych, żeby nauczyć się gatunków | Wysoki | ✅ DONE |
| US-1.2 | Jako uczeń, chcę filtrować rośliny po kategorii (drzewa, krzewy, grzyby...), żeby szybciej znaleźć potrzebną informację | Wysoki | ✅ DONE |
| US-1.3 | Jako uczeń, chcę wyszukać roślinę po nazwie, żeby szybko znaleźć konkretny gatunek | Wysoki | ✅ DONE |
| US-1.4 | Jako uczeń, chcę zobaczyć szczegóły rośliny (opis, siedlisko, ciekawostka), żeby zrozumieć gatunek | Wysoki | ✅ DONE |
| US-1.5 | Jako uczeń, chcę poznać warstwy lasu (gleba → niebo), żeby zrozumieć strukturę ekosystemu | Wysoki | ✅ DONE |
| US-1.6 | Jako uczeń, chcę czytać ciekawostki o zwierzętach i roślinach, żeby zapamiętać fakty do matury | Średni | ✅ DONE |

## ICP 2: Nauczyciel (Dorota, 42 lata)
> Nauczycielka biologii szukająca materiałów dydaktycznych online dla swoich uczniów.

| # | User Story | Priorytet | Status |
|---|---|---|---|
| US-2.1 | Jako nauczyciel, chcę mieć interaktywny materiał o lesie, żeby pokazać go uczniom na lekcji | Wysoki | ✅ DONE |
| US-2.2 | Jako nauczyciel, chcę żeby strona była responsywna, żeby działała na tablecie w sali lekcyjnej | Wysoki | ✅ DONE |
| US-2.3 | Jako nauczyciel, chcę mieć dostęp do ciekawostek o każdej warstwie lasu, żeby urozmaicić lekcję | Wysoki | ✅ DONE |
| US-2.4 | Jako nauczyciel, chcę żeby strona była darmowa i nie wymagała logowania, żeby uczniowie mogli z niej korzystać | Wysoki | ✅ DONE |
| US-2.5 | Jako nauczyciel, chcę mieć linki do źródeł naukowych, żeby weryfikować treści | Średni | ✅ DONE |

## ICP 3: Rekruter IT (Michał, 33 lata)
> Rekruter techniczny oceniający portfolio kandydata na stanowisko Junior Frontend Developer.

| # | User Story | Priorytet | Status |
|---|---|---|---|

| US-3.2 | Jako rekruter, chcę sprawdzić czy strona działa na mobile, żeby ocenić responsywność | Wysoki | ✅ DONE |
| US-3.3 | Jako rekruter, chcę zobaczyć złożone funkcje JS (filtry, modale, canvas), żeby ocenić poziom kandydata | Wysoki | ✅ DONE |
| US-3.4 | Jako rekruter, chcę sprawdzić kod źródłowy, żeby ocenić jakość i organizację | Wysoki | ✅ DONE |
| US-3.5 | Jako rekruter, chcę zobaczyć że strona jest szybka i dostępna, żeby ocenić dbałość o UX | Średni | ✅ DONE |
